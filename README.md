# 波波拳 - 完整游戏项目

## 项目结构

```
bobo-fist/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式
├── js/
│   ├── game.js         # 核心游戏逻辑
│   ├── characters.js   # 角色数据
│   ├── ai.js           # AI对手
│   └── ui.js           # 界面控制
└── assets/
    └── (图片资源占位)
```

## 游戏特性

- ✅ 充气蓄力系统（波波槽）
- ✅ 三大绝招释放
- ✅ 4个独特角色
- ✅ AI对手（3种难度）
- ✅ 血量模式 + 标准模式
- ✅ 道具卡系统
- ✅ 连击系统
- ✅ 完整UI动画

## 运行方式

直接用浏览器打开 `index.html` 即可游玩。
